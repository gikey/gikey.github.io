发表于 2017-05-23 |

由于windows坑实在太多，这里只介绍osx和linux环境下的利用github pages 和hexo博客搭建，有兴趣可以去windows下试试。这里对于Hexo只做最基本的介绍具体使用和配置请查看官网。

前提

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git

配置

关于_config.yml的配置只介绍几个常用的：

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述
author 您的名字
language 网站使用的语言
url 网址
root 网站根目录
permalink 文章的永久链接格式
per_page 每页显示的文章量 (0 = 关闭分页功能)
pagination_dir 分页目录
theme 主题
deploy 部署部分设置

配置文件是yml的语法，很多时候部署不成功都是因为语法不对(冒号后面有个空格!!!!)
