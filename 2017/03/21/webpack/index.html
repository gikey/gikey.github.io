Webpack2 简单使用

发表于 2017-03-21 |

webpack是一个现代的 Javascript 应用程序的模块打包器(module bundler)。它可以将许多松散的模块按照依赖和规则打包成符合生产环境部署的前端资源。还可以将按需加载的模块进行代码分隔，等到实际需要的时候再异步加载。webpack2相对webpack1还是有不小的区别，这里简单介绍下webpack2的基本使用。
