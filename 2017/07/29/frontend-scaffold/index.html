安装配置 charles

charles是mac端的一款截取与分析网络请求的工具，在网络开发中使用其作分析，可以大大提高我们的开发效率。charles 是一款付费软件，一般可以试用30天。charles 官网 https://www.charlesproxy.com/

代理手机上的网络请求

顶部菜单栏 -> Proxy -> Proxy Settings -> Proxies -> 勾上 Enable transparent HTTP proxying

为了把手机的请求转发到 PC 上，我们要保证手机和 PC 是在同一个局域网的。所以他们要连接同一个 Wi-Fi 。
