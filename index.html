gulp

发表于 2017-05-23

Gulp.js 是一个自动化构建工具，开发者可以使用它在项目开发过程中自动执行常见任务。Gulp.js 是基于 Node.js 构建的，利用 Node.js 流的威力，你可以快速构建项目并减少频繁的 IO 操作。Gulp.js 源文件和你用来定义任务的 Gulp 文件都是通过 JavaScript（或者 CoffeeScript ）源码来实现的。gulp官网。

安装

1.安装Node.js

安装gulp之前需要先安装Node.js，window、osx下可以去官网下载相应版本直接安装。下面介绍linux和osx下载源码安装：

1.首先进到下载页面,下载Source Code的tar.gz包。(以node-v4.4.4.tar.gz为例)
2.解压node-v4.4.4.tar.gz

3.进入解压的文件夹，执行下面命令：

$ cd node-v4.4.4/
$ ./configure

  1. 直接执行make命令（编译时长根据计算机性能）

    $ make
    5.编译完成后执行安装命令：

    $ sudo make install

安装完成后执行命令：

$ node -v
v4.4.4

安装成功！

另外也可以下编译好的二进制文件进行安装 tar --strip-components 1 -xzvf node-v* -C /usr/local

2.安装gulp

全局安装gulp(非必须)：

$ sudo npm install --global gulp

查看npm安装的packpages: npm list -g –depth=0

安装完毕后可以通过：

$ gulp -v

查看gulp是否安装成功。

使用

#####基本使用
1.作为项目的开发依赖（devDependencies）安装：

$ npm install --save-dev gulp

2.在项目根目录下创建一个名为 gulpfile.js 的文件：

var gulp = require('gulp');
gulp.task('default', function() {
    console.log('hello gulp!');  // 将你的默认的任务代码放在这
});

3.运行gulp：

$ gulp

会在控制台打印出 “hello gulp!”

#####扩展插件

  • css代码压缩

安装gulp-csso：

$ npm install --save-dev gulp-csso

gulpfile.js文件：

var gulp = require('gulp')，
    csso = require('gulp-csso');
gulp.task('default', function() {
    return gulp.src('lib/css/*.css')
      .pipe(csso())
      .pipe(gulp.dest('build/css');
});

运行gulp即可将lib/css下的所有css文件压缩后输出到build/css下。也可以使用gulp-clean-css进行css压缩，但是必须是在.pipe(gulp.dest(‘build/css’)后执行。

  • js代码压缩

安装gulp-uglify：

$ npm install --save-dev gulp-uglify

gulpfile.js文件：

var gulp = require('gulp')，
    uglify = require('gulp-uglify');
gulp.task('default', function() {
    return gulp.src('lib/js/*.js')
      .pipe(uglify())

// .pipe(uglify({license: some})) 不压缩 /! xxx/信息
.pipe(gulp.dest(‘build/js’);
});
运行gulp即可将lib/js下的所有js文件压缩后输出到build/js下。

  • 代码合并

安装gulp-concat：

$ npm install --save-dev gulp-concat

gulpfile.js文件：

var gulp = require('gulp'),
    concat = require('gulp-concat')；
gulp.task('default', function() {
    return gulp.src('lib/js/*.js')
      .pipe(concat('all.js'))
      .pipe(gulp.dest('build/js'));
});

这会将lib/js下的所有js文件合并成all.js输出到build/js下。
也可以将指定的文件合并：

var gulp = require('gulp'),
    concat = require('gulp-concat')；
gulp.task('default', function() {
    return gulp.src(['lib/js/file1.js','lib/js/file2.js'])
      .pipe(concat('all.js'))
      .pipe(gulp.dest('build/js'));
});

这会将指定的这两个文件合并成all.js输出到build/js下。
gulpfile.js文件：

var gulp = require('gulp'),
    concat = require('gulp-concat')；
gulp.task('default', function() {
    return gulp.src(['lib/js/file1.js','lib/js/file2.js'])
      .pipe(concat({ path: 'all.js', stat: { mode: 0666 }}))
      .pipe(gulp.dest('build/js'));
});

这样将会合并两个文件。

  • less和sass预编译

安装gulp-less：

$ npm install --save-dev gulp-less

安装gulp-sass：

$npm install --save-dev gulp-sass

在gulp中less和sass的使用方法基本一致，这里只介绍sass的使用。

gulpfile.js文件：

var gulp = require('gulp'),
    sass = require('gulp-sass')；
gulp.task('default', function() {
    return gulp.src('lib/css/style.scss')
      .pipe(sass().on('error', sass.logError))
      .pipe(gulp.dest('build/js'));
});

运行gulp即可将style.scss文件编译为style.css文件。

也可以设置输出代码样式（nested、expanded、compact、compressed）：

.pipe(sass({outputStyle: 'compressed'}).on('error', sass.logError))

sass还可以使用compass的mixins

安装compass-sass-mixins：

$ npm install --save-dev compass-sass-mixins

即可在相关的sass文件中引入就可以使用了，如style.scss：

@import 'node_modules/compass-sass-mixins/lib/compass';

div {
    width: 5rem;
    height: 5rem;
    background-color: #000;
    @include border-radius(50%);
}
  • webserver

webserver可以启动一个web服务，这样可以使用ajax、自动刷新等功能了。

安装gulp-webserver：

$ npm install --save-dev gulp-webserver

webserver加上livereload就能实现自动刷新功能：

另一个监视scss文件改变就自动刷新页面的例子,gulpfile.js文件：

var gulp = require('gulp'),
    sass = require('gulp-sass'),
    webserver = require('gulp-webserver');

gulp.task('default',['watch','webserver','sass']);

gulp.task('watch',function() {
 gulp.watch('sass/*.scss',['sass']);
});

gulp.task('sass', function() {
  return gulp.src('sass/*.scss')
   .pipe(sass({outputStyle: 'compact'}))
   .pipe(gulp.dest('css'));
});

gulp.task('webserver',function(){
   return gulp.src('./')
   .pipe(webserver({
         // port: 5050, 设置监听端口号
         livereload: true,
         directoryListing: true,
         open: true
     }));
});

这里需要在浏览器里安装livereload这个插件并启动，否则不会自动刷新，使用过程中不能使用ShadowsocksX否则webserver不能启动。

  • 检测js语法

安装gulp-jshint：

$ npm install --save-dev gulp-jshint

gulpfile.js文件：

var jshint = require('gulp-jshint');

gulp.task('jshint',function(){
  return gulp.src('js/*.js')
    .pipe(jshint())
    .pipe(jshint.reporter('default'));
});

  • 合图

安装spritesmith：

$ npm install --save-dev gulp.spritesmith

gulpfile.js文件：

var gulp=require('gulp'),
    spritesmith=require('gulp.spritesmith');

gulp.task('default', function () {
   return gulp.src('images/*.png')//需要合并的图片地址
     .pipe(spritesmith({
        imgName: 'sprite.png',//保存合并后图片的地址
        cssName: 'css/sprite.css',//保存合并后对于css样式的地址
        padding:5,//合并时两个图片的间距
        algorithm: 'binary-tree',//Algorithm 有四个可选值分别为top-down、left-right、diagonal、alt-diagonal、binary-tree
     }))
    .pipe(gulp.dest('dist/images'));
});

详情可参考gulp多张图片自动合成雪碧图这篇文章，作者写得很详细。
